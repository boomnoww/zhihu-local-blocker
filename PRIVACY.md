# 隐私政策

最后更新：2026-06-18

“知乎本地屏蔽助手”是一款在用户浏览器本地运行的开源浏览器扩展。

## 数据处理

- 扩展不会建立开发者服务器，也不会把用户数据发送给开发者或第三方分析服务。
- 本地黑名单、扩展设置、官方同步队列、断点续传状态和必要任务日志仅保存在浏览器提供的 `chrome.storage.local` 中。
- 扩展只在 `https://www.zhihu.com/*` 页面运行，并使用用户当前已登录的知乎会话完成页面识别及用户主动触发的操作。
- 扩展不会读取或保存知乎账号密码、支付信息、浏览器历史记录或其他网站内容。
- 详细调试日志默认关闭；用户可在设置页的“高级诊断”中临时开启、查看、导出或清空日志。

## 数据共享与出售

扩展不出售、出租、共享或用于广告分析任何用户数据。

## 数据删除

用户可以在扩展设置页移除本地名单、清空日志，也可以通过卸载扩展删除由扩展保存的全部本地数据。

## 权限用途

- `storage`：保存本地名单、设置、同步队列和任务状态。
- `tabs`：查找或打开知乎标签页，并向知乎页面发送用户主动触发的同步命令。
- `scripting`：扩展更新后，在已有知乎标签页中重新注入本扩展脚本。
- `https://www.zhihu.com/*`：仅用于知乎网页端的内容识别和用户主动触发的操作。

## 联系与变更

隐私政策变更会在项目仓库中公布。问题或建议请通过 GitHub Issues 提交。
