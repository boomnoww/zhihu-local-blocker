# 知乎本地屏蔽助手

一个适用于Microsoft Edge的浏览器扩展。可以在知乎网页端标注并折叠隐藏指定用户的回答和评论。
在知乎网站中浏览时，即使你已经拉黑了某些用户，但依然能偶尔在回答中或评论中看到对方的言论。本插件可以直接折叠被拉黑用户的回答和评论，同时提供一键拉黑特定回答答主和所有为该回答点赞用户的功能，可以迅速将与自己认知立场相悖的群体填充进黑名单中，从而避免多次重复见到令自己厌恶的言论。

> 本项目是非官方开源扩展，与知乎无隶属、合作或认可关系。

## 主要功能

- 从知乎官方黑名单页面导入用户到扩展本地名单。
- 通过知乎用户主页 URL 手动加入本地黑名单。
- 在回答和评论中按稳定用户主页 token 识别用户，不使用显示名称匹配。
- 支持折叠并标注、仅标注、涂黑、完全隐藏四种显示方式。
- 在回答旁一键处理“答主”或“答主 + 赞同者”。
- 支持批量任务断点续传；遇到知乎验证或风控时保存进度。
- 支持本地名单 JSON 导入、导出、搜索和移除。
- 支持补全尚未完成的知乎官方拉黑同步。
- 详细调试日志默认关闭，可在“高级诊断”中临时开启。

## 安装

### 从商店安装

商店链接将在首次发布通过后补充。

### 开发者模式安装

1. 下载源码并解压。
2. Edge 打开 `edge://extensions/`。
3. 开启“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择包含 `manifest.json` 的项目根目录。

更新源码后，需要在扩展管理页点击“重新加载”，并刷新已经打开的知乎页面。

## 使用方法

### 导入已有知乎官方黑名单

1. 登录知乎网页端并打开 `https://www.zhihu.com/settings/filter`。
2. 点击页面顶部由扩展插入的“同步本页及后续黑名单”。
3. 等待任务面板提示完成。

### 手动加入用户

1. 打开扩展设置页。
2. 在“手动加入”中每行填写一个知乎用户主页 URL。
3. 点击“加入本地黑名单”。

### 处理答主和赞同者

1. 打开包含目标回答的知乎页面。
2. 点击答主旁的“拉黑答主”或“拉黑答主 + 赞同者”。
3. 扩展先写入本地名单，再按设置尝试知乎官方拉黑。

部分赞同者数据只有知乎内部 member id，没有公开主页 token。这些用户仍可保留在本地名单中，但会标记为 `unavailable`，不会向无效官方接口重复请求。

### 调试日志

设置页底部的“高级诊断”默认收起，“记录详细调试日志”默认关闭。正常使用时只保留必要的任务结果、错误和验证暂停信息。

## 自动化边界

- 扩展不会绕过验证码、安全验证、登录限制或访问控制。
- 官方拉黑同步是尽力行为，依赖知乎当前网页接口，可能因接口变化、账号状态或风控而失败。
- 遇到验证或风控时，扩展会保存批量任务进度并暂停；完成验证后可继续。
- 即使官方同步失败，本地名单仍会保留并继续用于页面折叠。
- 请控制批量操作频率，并遵守知乎服务条款和适用法律。

## 隐私与权限

扩展没有后端服务，不包含统计分析或广告 SDK。黑名单、设置、队列、任务状态和必要日志保存在 `chrome.storage.local`，不会随源码或发布包上传。

权限用途：

- `storage`：保存本地名单、设置、同步队列和任务状态。
- `tabs`：查找或打开知乎标签页，并发送用户主动触发的任务命令。
- `scripting`：扩展更新后，在已有知乎标签页中重新注入扩展脚本。
- `https://www.zhihu.com/*`：只在知乎网页端识别页面内容和执行用户主动触发的操作。
